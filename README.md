## Getting Started

To get the project up and running locally, follow these steps:

```bash
# Clone the repository
# git clone https://github.com/yourusername/tool-management-backend.git

# Change directory
# cd tool-management-backend

# Install dependencies
# mvn install

# Run the application
# mvn spring-boot:run
```
